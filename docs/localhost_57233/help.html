Workgroup

APIDescription
PUT api/workgroups/{workgroupId}/workstations/{workstationId}/sharing

Enable of disable sharing capabilities for a particular workstation in a workgroup
PUT api/workgroups/{workgroupId}/sharing

Enables or disables sharing for the whole workgroup
POST api/workgroups/{workgroupId}/workstations/{workstationId}

Adds a workstation to the workgroup Assumes the workgroup is in progress, and the workstation is available
DELETE api/workgroups/{workgroupId}/workstations/{workstationId}

Will remove a given workstation from the workgroup Assumes workgroup is still in progress, and that workstation is currently in the workgroup
GET api/workgroups/{workgroupId}/workstations

Returns all of the workstations that are currently a member of the workgroup
POST api/workgroups

Creates a new workgroup Assumes all of the workstations are currently available, and that the user making the workgroup, doesn't have any currently running or scheduled workgroups
