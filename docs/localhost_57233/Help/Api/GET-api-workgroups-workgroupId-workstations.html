GET api/workgroups/{workgroupId}/workstations

Returns all of the workstations that are currently a member of the workgroup

Request Information

URI Parameters

NameDescriptionTypeAdditional information
workgroupId

The target workgroup

 integer

Required

Body Parameters

None.

Response Information

Resource Description

NameDescriptionTypeAdditional information

 boolean

None.

 date

None.
Identifier

The IT issued identifier for the workstation

 string

None.
TopXRoomOffset

The percentage offset of the workstation relative to the top of an image of a given room - x axis

 decimal number

None.
TopYRoomOffset

The percentage offset of the workstation relative to the top of an image of a given room - y axis

 decimal number

None.
Available

Indicates if a workstation is already part of a workgroup or otherwise unavailable for use (note that this does not mean a workstation is offline)

 boolean

None.

 integer
