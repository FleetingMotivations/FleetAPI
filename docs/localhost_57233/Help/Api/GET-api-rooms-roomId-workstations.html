GET api/rooms/{roomId}/workstations

Returns a collection of all of the workstations for a given room

Request Information

URI Parameters

NameDescriptionTypeAdditional information
roomId

integer

Required

Body Parameters

None.

Response Information

Resource Description

Collection of WorkstationModel
NameDescriptionTypeAdditional information
LastSeen

The last time the workstation announced itself as being available (talking to the WebService)

 date

None.
Identifier

The IT issued identifier for the workstation

 string

None.
TopXRoomOffset

The percentage offset of the workstation relative to the top of an image of a given room - x axis

 decimal number

None.
TopYRoomOffset

The percentage offset of the workstation relative to the top of an image of a given room - y axis

 decimal number

None.
Available

Indicates if a workstation is already part of a workgroup or otherwise unavailable for use (note that this does not mean a workstation is offline)

 boolean

None.
Colour

The colour associated with this workstation within its room

 string

None.
Name

The display name of the item

 string

None.
Id

The database Id of the model

 integer

None.
