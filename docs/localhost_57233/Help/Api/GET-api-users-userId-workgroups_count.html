GET api/users/{userId}/workgroups?count={count}

Returns a list of prior workgroups that a user conducted

Request Information

URI Parameters

NameDescriptionTypeAdditional information
userId

The user that conducted the workgroups

 integer

Required
count

The amount of listings to retrive

 integer

Default value is 5

Body Parameters

None.

Response Information

Resource Description

Collection of WorkgroupListingModel
NameDescriptionTypeAdditional information
Room

The primary room from which the workgroup took place

 GenericItemModel

None.
Started

The time at which the workgroup started

 date

None.
Id

The database Id of the model

 integer

None.
