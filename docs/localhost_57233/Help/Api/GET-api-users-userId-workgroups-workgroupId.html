GET api/users/{userId}/workgroups/{workgroupId}

Returns a workgroup summary belonging to a given user The summary is based on the state of the workgroup at the end of the session

Request Information

URI Parameters

NameDescriptionTypeAdditional information
userId

The user that created the workgroup

 integer

Required
workgroupId

The id of the workgroup to retrieve

 integer

Required

Body Parameters

None.

NameDescriptionTypeAdditional information

The time at which the workgroup ended

 date

None.
Duration

The amount of minutes the workgroup ran for

 integer

None.
AllowedApplications

The applications that were permitted to run during the workgroup

 Collection of GenericItemModel

None.
Room

The primary room from which the collaboration session took place

 GenericItemModel

None.
Workstations

The workstations that were involved in the collaboration session when it ended

 Collection of WorkstationModel

None.
